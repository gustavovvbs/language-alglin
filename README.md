# language-alglin

Esse projeto estuda o uso de representações vetoriais para palavras.

## Como rodar

1. Instale as dependências:
```bash
pip install -r requirements.txt
```

2. Rode o treinamento e/ou plote o cluster de sentimento:
```bash
python src/train.py
```
